Modular JavaScript

Constructor Functions

Function that defines the properties and methods of an object:

const Dice = function(sides=6){
    this.sides = sides;
    this.roll = function() {
        return Math.floor(this.sides * Math.random() + 1)
    }
}

We can now create an instance of the dice constructor function using the new operator.

const redDice = new Dice();
<< Dice { sides: 6, roll: [Function] }

Prototypes

Default properties and methods can be added to the class declaration. In additio, you can augment the class with extra methods and properties after it has been created using the prototype property of the class.

Adding a Method Example

Turtle.prototype.attack = function(){
return `Feel the power of my ${this.weapon}!`;
}
<< [Function]

Inheritance

The previous example demonstrates inheritance by inheriting properties and methods from the prototype.

Mixins

A mixin is a means to add properties and methods of some objects to another object without using inheritance. It allows more complex objects to be created by ‘mixing’ basic objects together.

Basic Functionality

Object.assign()

Example

const a = {};

const b = { name: 'JavaScript' };

Object.assign(a,b);
<< { name: 'JavaScript' }

a.name
<< 'JavaScript'

Borrowing Methods from Prototypes

It’s possible to borrow methods from objects without having to inherit all their properties and methods by making a reference to the function that you want to borrow (without parentheses so that it isn’t invoked).

Example:

const fly = superman.fly;
<<

fly.call(batman);
<< Up, up and away! Batman soars through the air!
